Android移动客户端

Megumin

Megumin is a Albireo client for Android devices.It allows connecting to your own Albireo server.

Megumin designed to browse and play bangumi on Albireo server, so it does not support some features (e.g. edit or manage download tasks).

Megumin is open source software, you can find source code in https://github.com/RoyaAoki/Megumin

Mana

Mana, an unofficially client for Albireo, is ready to help you search, index and enjoy videos that saved on the Albireo.

Before getting started, we recommend you get familiar with Albireo first and prepare the URL for a specific Albireo server as service provider

Hope you enjoy!

Mana is open source software, you can find all the code in

https://github.com/WindFi/mana
